flutter开发环境搭建

flutter 环境搭建

  • 配置JavaJDK

    • 下载地址 javaSDKopen in new window
    • 下载JavaSDK 并且安装jdk 安装完成后高版本没有jre 目录下执行 bin\jlink.exe --module-path jmods --add-modules java.desktop --output jre
    • 配置环境变量 JAVA_HOME 'C:\Program Files\Java\jdk-22
    • 配置 %JAVA_HOME%\bin %JAVA_BOME%\jre\bin
    • 命令行 java javac 输出信息安装成功

  • 安装Android Studio

    • 下载地址 Android Studio下载地址open in new window
    • 安装完成后 安装dart flutter plgu
    • 下载汉化包 汉化包地址open in new window

  • 下载Flutter SDK

    • 下载flutter SDK 地址 ： flutter地址open in new window
    • 配置flutter 环境变量 path 新增 C:\Program Files\flutter\bin
    • flutter --version 检测成功

  • 配置Flutter国内镜像 (可以不需要-----------)

    • 配置系统变量 https://flutter.cn/community/china
    • 新建系统环境变量

  • 运行flutter doctor 命令 检测环境是否配置成功，根据提示安装对应软件

    • 报错按照提示安装

  • 打开Andriod studio 安装flutter Dart 插件

  • 运行flutter项目

    • flutter run [-d] all / flutter run
    • flutter 第一次运行 会卡在gradle下载这一部分上 所以需要修改一个下载地址 andriod\gradle\wrapper\gradle-wrapper 中的
        distributionBase=GRADLE_USER_HOME
    distributionPath=wrapper/dists
    zipStoreBase=GRADLE_USER_HOME
    zipStorePath=wrapper/dists
    #distributionUrl=https\://services.gradle.org/distributions/gradle-7.6.3-all.zip
    distributionUrl=https\://mirrors.cloud.tencent.com/gradle/gradle-7.6.3-all.zip
    • 修改这里其实还是会卡住，其他教程修改仓库源等试了一下 都没有效果，最后反而会报错，这里第一次会卡2-3个小时左右

VsCode 连接安卓模拟器

  • 安装夜神模拟器
  • 打开模拟器 开发者选项 -- USB调试打开
  • 模拟器文件目录下bin 下 执行 adb connect 127.0.0.1:62001 或者 nox_adb.exe connect 127.0.0.1:62001
  • 打开Vscode 右下角 devices 选择模拟器设备
