Dart变量、常量、打印、注释

  • 变量 | Dart语言中变量基本与Js相同，var a = 1; 同时还支持 String a = '1'; 但是不同与js的是，变量类型不可修改，重新赋值只能赋值为同一类型的值
  • 常量 | 支持 const 与 final 两种方式创建
  • 打印 | print()
  • 注释 | 与Js中相同 // 双反斜号
void main() {
  var str1 = 'dart'; 
  String str2 = 'hello world'; // 字符串类型
  var num1 = 123;
  int num2 = 456; // 创建整型

  const num3 = 789; // 常量
  final num4 = 111; // 常量

  print(num1);
  print(num2);
  print(str1);
  print(str2);
  // hello world
}

class的私有属性

私有属性只能在class内部读取，而生成的实例是无法读取的。_name , 私有属性与JS相同

  // dart 中的class 私有变量 私有方法
  // class 必须 从其他模块引入才会 实例不可访问 私有变量 私有方法

  class Animal {
    String _privateVar = '私有变量';
    String publicVar = '公有变量';
    Animal() {
      print('我是Animal 构造函数');
    }
    // ignore: unused_element
    void _privateMe() {
      print('我是私有方法');
    }

    void publicMe() {
      print('我是公有方法');
    }
  }

  void main() {
    Animal dog = new Animal();
    print(dog._privateVar);
    print(dog.publicVar);
    dog._privateMe();
  }

class继承

  • class的继承 extends 关键字
  • class构造函数 constructor([options]) : super([options])
  • class方法复写

class Animal {
  String weight = '';

  Animal(weight) {
    this.weight = weight;
  }

  void getWeight() {
    print('Animal');
  }
}

class Dog extends Animal {
  String color = '';
  String weight = '';
  Dog(weight, color) : super(weight) {
    this.color = color;
    this.weight = weight;
  }
  // 方法复写
  @override
  void getWeight() {
    print('dog');
  }
}

void main() {}

Dart中的抽象class以及接口

抽象class abstract Class

定义抽象类 继承 抽象类


abstract class Animal {
  late String weight;
  eat();
}

class Dog extends Animal {
  // extends 实现抽象类必须完成 抽象类中方法
  eat() {}
}

接口 class implements

接口与抽象类的不同

// 抽象类
abstract class Animal {
  late String weight;
  eat();
}

class Dog extends Animal {
  // extends 实现抽象类必须完成 抽象类中方法
  eat() {}
}

class Cat implements Animal {
  // 接口 必须全部都完成 抽象类中的方法以及属性
  @override
  late String weight;

  @override
  eat() {
    // TODO: implement eat
    throw UnimplementedError();
  }
}
